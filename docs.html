  • The original 6S paper by Vermote et al., plus the original 6S manuals
  • A list of published papers which cite Py6S - so you can see what other people have done with it
  • The Py6S category on my blog, which has various articles about how to use Py6S
  • If you want to see how Py6S works on the inside, then all of the code is freely available on Github
